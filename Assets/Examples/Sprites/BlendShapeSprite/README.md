# AddBlendShape Sample

## Demonstrates

AddBlendShape and related API for sprites. The window will generate SpriteShapes and frames for any selected sprite.

## Example Uses

-Deformable sprite terrain
-Damage indicator

## How To Use

Open: Tools > Sprite Blend Shape Authoring

Drag the "Metal Box" sprite in the Sprites subdirectory into the target. 

The temporary object "BlendShape Preview" will be generated in the Hierarchy, with the sprite bones as its children. 

Choose a Pose Name to create a new Sprite Blend Shape.

Set the Frame Weight for the current blend shape.

In the Scene View: move one or more bones and set a higher Frame Weight every time you hit the button.
